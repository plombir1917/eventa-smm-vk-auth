# Эвента - SMM бот для Telegram

Проект представляет собой веб-приложение для авторизации пользователей через ВКонтакте, интегрированное с SMM ботом для Telegram.

## Функциональность

- Авторизация через ВКонтакте (VK ID)
- Адаптивный дизайн
- Интеграция с Telegram ботом

## Технологии

- Vue.js 3
- VK ID SDK
- Nuxt.js

## Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone [url-репозитория]
cd eventa-smm-vk-auth
```

2. Установите зависимости:

```bash
npm install
```

3. Создайте файл `.env` и добавьте необходимые переменные окружения:

```env
VK_APP_ID=ваш_id_приложения
REDIRECT_URL=ваш_url_редиректа
```

4. Запустите проект в режиме разработки:

```bash
npm run dev
```

5. Для сборки проекта:

```bash
npm run build
```

## Конфигурация

### VK ID SDK

В файле `app.vue` настройте параметры VK ID SDK:

```javascript
VKID.Config.init({
  app: YOUR_APP_ID,
  redirectUrl: YOUR_REDIRECT_URL,
  scope: 'email phone',
});
```
